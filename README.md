# P3_AP-IA

Proyecto estructurado para producción siguiendo la habilidad `setup-prod-ia`.

## Entorno Virtual

El entorno virtual de este proyecto se encuentra contenido dentro del propio proyecto en el directorio `venv/`.

Para activarlo en Windows (PowerShell):
```powershell
.\venv\Scripts\Activate.ps1
```

O en la línea de comandos de Windows (cmd):
```cmd
.\venv\Scripts\activate.bat
```

## Estructura
- `data/`: Datos crudos (raw) y procesados.
- `src/`: Código fuente principal.
- `models/`: Modelos entrenados e iteraciones.
- `playground/`: Cuadernos (notebooks) y pruebas.
- `tests/`: Pruebas automatizadas.
